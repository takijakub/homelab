# homelab
I started homelabbing around 2023. While out on a bike ride with a friend, he mentioned he had an old PC sitting around - and offered it to me to experiment with. My journey began with an Ubuntu server and basic SSH access. Soon after, I was running Proxmox with multiple virtual machines, exploring different systems and learning a lot of new concepts along the way.

Now, a few years in, I'd like to open up what I've built - share the configs, document the decisions I’ve made along the way, and maybe give someone else the nudge to start their own setup.
